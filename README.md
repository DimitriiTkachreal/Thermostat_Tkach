## Опис проєкту
Цифровий термостат реального часу на базі 8-бітного мікроконтролера STM8. Проєкт є функціональним аналогом популярного модуля W1209. Прошивка написанау у стилі Bare-Metal мовою C (стандарт C89, компілятор COSMIC) без використання сторонніх бібліотек. Розроблено в рамках стажування Ajax (перший проєкт).

### Апаратна частина (Hardware)
* **MCU:** STM8S103F3P6 (16 MHz, 1KB RAM, 8KB Flash, 640 bytes EEPROM).
* **Живлення:** USB Type-C (5V). Логіка мікроконтролера живиться від 3.3V через лінійний стабілізатор AMS1117-3.3. Для зменшення нагріву стабілізатора силова частина (реле) та периферія живляться безпосередньо від 5V.
* **Узгодження рівнів:** Оскільки МК працює на 3.3V, а дисплей та UART на 5V, на лініях даних встановлено двонаправлені конвертери логічних рівнів.
* **Сенсор:** NTC термістор 10kΩ (B3950) + резистор 10kΩ (дільник напруги). Підключений до піна **PC4** (АЦП канал 2, вхід налаштовано як Floating).
* **Дисплей:** 4-розрядний 7-сегментний LED-індикатор з драйвером TM1637. Підключений до **PC6 (DIO)** та **PC7 (CLK)**. Піни налаштовані в режим Open-Drain (швидкість 10 МГц) із зовнішніми підтяжками.
* **Керування:** 3 тактові кнопки (SET - **PC5**, UP - **PD2**, DOWN - **PD3**). Налаштовані як Floating входи, оскільки на платі розведені зовнішні підтягуючі резистори 10kΩ до 3.3V.
* **Вихідний каскад:** Електромеханічне реле (керується через пін **PC3**, режим Push-Pull).
* **Дебаг:** Вбудований USB-UART міст CH340C. Прошивання мікроконтролера, через ST-Link v2 (інтерфейс SWIM).

## Інструкція з користування
Керування здійснюється кнопками: `SET`, `UP`, `DOWN`.

### Головний екран
У робочому режимі дисплей відображає поточну температуру в приміщенні. Якщо температура виходить за межі встановлених лімітів, МК подає логічну одиницю на реле для увімкнення нагріву/охолодження (загоряється відповідний індикатор). Для швидкої зміни цільової температури коротко натисніть `SET`. Якщо датчик пошкоджено або температура виходить за межі вимірювання, пристрій переходить у захист і відображає `Err` або `ALrM`.

### Меню налаштувань
Для входу у глибокі налаштування утримуйте кнопку **SET** більше **1 секунди**.
* **Навігація:** `UP` / `DOWN` перемикають параметри (P0...P6).
* **Редагування:** Натисніть `SET` для вибору, змініть значення кнопками `UP`/`DOWN`, натисніть `SET` для збереження в енергонезалежну пам'ять.
* **Вихід:** Утримуйте `SET` або зачекайте 10 секунд (автовихід).

#### 📋 Карта параметрів
| Код | Назва параметра | Опис функції | Фізичний діапазон МК | Заводське |
|:---:|:----------------|:-----------------------------------|:--------------------------------:|:---------:|
| **P 0** | **Mode** | Режим роботи термостата | `0` (Нагрів) / `1` (Охолодження) | `0` |
| **P 1** | **Hysteresis** | Гістерезис (різниця вкл/викл) | 0.1 ... 15.0 °C | 2.0 °C |
| **P 2** | **Max Limit** | Обмеження максимальної температури | -20.0 ... 120.0 °C | 80.0 °C |
| **P 3** | **Min Limit** | Обмеження мінімальної температури | -20.0 ... 120.0 °C | -20.0 °C |
| **P 4** | **Calibration**| Калібрування похибки датчика | -10.0 ... +10.0 °C | 0.0 °C |
| **P 5** | **Delay** | Затримка ввімкнення (захист компресора)| 0 ... 10 хв | 0 хв |
| **P 6** | **Alarm** | Температура аварійного вимкнення | -20.0 ... 120.0 °C | 85.0 °C |
